= Past simple 
(простое прошедшее время)

____ + V2, ed.

I worked a lot yesterday.

I got a lot of money.

== Negation

did + not = didn't

I didn't work a lot yesterday.

I didn't get a lot of money.

I did work a lot.

Did you see him yesterday? - Yes, I did. / No, I didn't.

What did you do?

== To be 

~~ был

To be (was/were)

I, he, she, it - was

you, we, they - were

I was busy.

I wasn't busy.

Were you busy? - Yes, I was.
